# BroadFire Ai - Research Lab Landing Page

An interactive, high-performance landing page for a research lab specializing in Embodied Agents, Spatial Intelligence, and Edge AI. This project features advanced procedural WebGL shaders using Three.js, a custom design system with Tailwind CSS, and a responsive React architecture.
